# Sistema de Autoescola

Projeto desenvolvido em Java para simular o funcionamento de uma autoescola.

## Funcionalidades

* Cadastro de alunos
* Agendamento de aulas teoricas e praticas
* Cancelamento de aulas
* Realizacao de exame
* Exibicao de resumo dos alunos

## Estrutura do Projeto

* Aluno.java
* Instrutor.java
* Veiculo.java
* AutoEscola.java
* Main.java

## Como executar

1. Compilar os arquivos:
   javac *.java

2. Executar o programa:
   java Main
